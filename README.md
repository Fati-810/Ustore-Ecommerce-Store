# 🛒 UStoreApp – WordPress eCommerce Website (Portfolio Project)

UStoreApp is a modern **WordPress-based eCommerce website** built as a **portfolio project** to demonstrate my skills in WordPress development, WooCommerce setup, UI customization, and responsive design.  
This project represents a **fully designed online store interface**, converted into a **static demo** for showcasing on GitHub Pages.

⚠️ This repository does NOT contain a full WordPress backend. It is a static version created only for demonstration purposes.

## Features:

🖥️ Responsive Homepage Design with hero slider.

🛍️ WooCommerce Product Listings (Shop & Categories).

📦 Individual Product Pages (UI demonstration).

🛒 Add to Cart & Cart Page UI Flow.

📰 Blog Section with categories and posts layout.

📱 Mobile-Friendly & Fully Responsive Design.

🧭 Modern Navigation Menu with dropdown support.

🎨 Clean & Professional eCommerce UI.

⚡ Fast-loading Static Demo hosted on GitHub Pages.

## Pages Included:
 Home
 
 Shop
 
 Product Detail Pages
 
 Blog (Categories & Posts UI)
 
 About
 
 Contact Us
 
 Cart (UI Demo)

## Technical Overview: (Behind the Scenes)

**Platform:** WordPress (CMS-based architecture)

**eCommerce Engine:** WooCommerce (products, cart, and shop UI)

**Theme Framework:** Hestia (customized layout & styling)

**Frontend Markup:** HTML5 (semantic structure)

**Styling:** CSS3 & Bootstrap (responsive grid system)

**Interactivity:** JavaScript & jQuery

**Icons:** Font Awesome

**Responsiveness:** Mobile-first & cross-browser compatible

**Deployment:** Static export hosted on GitHub Pages

**Purpose:** Portfolio showcase (UI/UX & frontend demonstration)
